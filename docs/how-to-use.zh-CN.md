# 如何使用 Skipisode

## 如何打开控制面板
1. 打开一个 YouTube 视频播放页面。
2. 在页面中找到圆形按钮，它通常显示在屏幕右下角。
![button](<../screenshots/floating button.png>)
3. 点击该按钮以打开控制面板。
![control panel](<../screenshots/control-panel-cn.png>)

## 如何设置片头跳过规则
1. 打开控制面板。
2. 播放视频，直到片头结束的位置（如果这样更方便，你也可以先暂停视频再设置时间点）。
3. 点击 `片头结束于` 卡片中的 `设置`，保存片头跳过时间点。
![intro card](<../screenshots/intro-card-cn.png>)
4. 如果当前视频属于播放列表，Skipisode 会询问你是否要将该规则应用到整个播放列表。你可以选择 `应用到全部剧集s` 或 `仅此一集`。
![apply to playlist demo](<../screenshots/playlist-cn.png>)

## 如何设置片尾跳过规则
1. 打开控制面板。
2. 播放视频，直到片尾或演职员表开始的位置。
3. 点击 `片尾结束于` 卡片中的 `设置`，保存片尾跳过时间点。
![outro card](<../screenshots/outro-card-cn.png>)
4. 如果当前视频属于播放列表，Skipisode 会询问你是否要将该规则应用到整个播放列表。你可以选择 `应用到全部剧集s` 或 `仅此一集`。

## 如何清除规则
1. 打开控制面板。
2. 在控制面板左下角找到 `清楚规则` 按钮。
3. 点击该按钮，然后选择清除仅当前这一集的规则，或整个播放列表的规则。
